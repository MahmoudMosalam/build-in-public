# Build Your Own Data Center  
# تعلم كيف تبني داتا سنتر شخصي… خطوة بخطوة، وبشكل عملي، وبدون الاعتماد على شركات الويب هوستنج

 مشروع مفتوح المصدر هدفه تمكين أي شخص من بناء داتا سنتر شخصي او لشركتك بشكل احترافي، وفهم كل طبقة من طبقات البنية التحتية بدل الاعتماد الكامل على شركات الاستضافة التقليدية.

هذا المشروع يتم بناؤه **علنًا** من خلال محاضرات أسبوعية كل يوم خميس على منصة **BrainsMangels.com**، مع توثيق كامل لكل خطوة هنا على GitHub.

https://brainsmingle.com/event-details/6268e3ae-0511-4957-88ae-7b5079f3a460

# أهداف المشروع:

- بناء مجتمع عربي واعٍ بالبنية التحتية الرقمية وطرق امتلاكها  
- تعليم الناس كيفية بناء داتا سنتر شخصي من الصفر  
- فهم الهاردوير، الشبكات، التخزين، الافتراضية، الحماية، والمراقبة  
- تقديم بديل عملي وفعّال عن شركات الويب هوستنج  
- مساعدة الافراد والـشركات على فهم البنية التحتية بعمق
